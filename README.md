# Portfolio

Meu portfolio pessoal, desenvolvido por mim para apresentar meus projetos.

[caducodedev.vercel.app](https://caducodedev.vercel.app/)

## Tecnologias

- NextJS
- Framer Motion
- SCSS
- RESEND API

### Páginas

- Home (Inicial)
- Projetos

O contato fica disponível no **footer**, presente em todas as páginas.

## Rodando Localmente

para rodar localmente basta clonar o repositório e e instalar as dependências

- git clone https://github.com/caadu999/portfoliov1.git
- cd portfoliov1
- npm install
- npm run dev
- Abrir http://localhost:3000 no navegador

## Variáveis de ambiente

O formulário de contato usa o Resend para envio de e-mails, então é necessário configurar a chave da API antes do build/execução.

- Crie um arquivo .env.local na raiz do projeto com:

RESEND_API_KEY=sua_chave_aqui

Você pode gerar uma chave gratuita criando uma conta em [resend.com](https://resend.com/).
