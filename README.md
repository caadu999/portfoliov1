# Portfolio

Meu portfolio pessoal, desenvolvido por mim para apresentar meus projetos.

[caducodedev.vercel.app](https://caducodedev.vercel.app/)

## Tecnologias

- NextJS
- Framer Motion
- SCSS
- RESEND API

### Páginas

- Home (Inicial)
- Projetos

O contato fica disponível no **footer**, presente em todas as páginas.

## Rodando Localmente

para rodar localmente basta clonar o repositório e e instalar as dependências

- git clone https://github.com/caadu999/portfoliov1
- cd portfolio
- npm install
- npm run dev
- Abrir http://localhost:3000 no navegador
